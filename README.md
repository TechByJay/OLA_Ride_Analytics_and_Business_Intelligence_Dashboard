# 🚖 OLA Ride Analytics & Business Intelligence Dashboard

## 1. 📌 Project Title
OLA Ride Analytics & Business Intelligence Dashboard

---

## 2. 📝 Brief One Line Summary
An interactive Power BI dashboard developed to analyze OLA ride booking data, revenue trends, cancellations, customer behavior, and vehicle performance for business decision-making.

---

## 3. 📖 Overview
The **OLA Ride Analytics & Business Intelligence Dashboard** is a data analytics project developed using **Power BI** to transform raw ride-booking data into meaningful business insights.  

This dashboard helps analyze:
- 📊 Overall ride bookings
- 💰 Revenue performance
- 🚘 Vehicle-wise performance
- ❌ Ride cancellation trends
- 👥 Customer booking behavior
- ✅ Booking success rates
- 💳 Payment method analysis

The project provides a complete business intelligence solution for understanding ride operations and improving decision-making.

---

## 4. ❗ Problem Statement
Ride-booking platforms generate massive amounts of operational data every day. However, extracting useful insights from raw booking data is difficult without proper visualization and analytics tools.

The main objective of this project is to:
- 📈 Analyze booking trends and ride performance
- 💵 Monitor revenue generation
- 🚫 Identify major cancellation reasons
- 🚗 Compare vehicle category performance
- 🧠 Improve business decisions using interactive dashboards

---

## 5. 📂 Dataset
The dataset used in this project contains approximately **100,000 ride booking records** related to OLA cab services.

### Dataset includes:
- Booking ID
- Customer ID
- Vehicle Type
- Booking Status
- Booking Value
- Ride Distance
- Payment Method
- Cancellation Reasons
- Ride Date
- Customer Ratings

## 7. ⚙️ Methods

The following steps were performed during the project development:

### 📥 Data Collection
Collected ride booking data from Excel dataset.

### 🧹 Data Cleaning
- Removed duplicate values
- Handled missing values
- Formatted columns properly
- Standardized categorical data

### 🔄 Data Transformation
Performed data preprocessing and created calculated measures using DAX.

### 📊 Dashboard Development
Created multiple interactive dashboard pages:
- Overall Analysis
- Vehicle Type Analysis
- Revenue Analysis
- Cancellation Analysis

### 📈 Data Visualization
Used:
- Pie Charts
- Bar Charts
- KPI Cards
- Tables
- Line Charts
- Filters & Slicers

## 8. 🔍 Key Insights

- 📌 Total bookings exceeded **100K rides**
- ✅ Most bookings were successfully completed
- 💳 Cash and UPI were the most preferred payment methods
- 🚘 Certain vehicle categories generated higher revenue
- ❌ Customer and driver cancellations impacted overall booking success
- 📅 Revenue trends varied across different dates and ride categories
